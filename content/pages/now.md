# Now

_Updated July 17, 2026._

I am building **CloakCode**, a way to observe and steer a live GitHub Copilot session from your
phone while keeping the workspace on the machine where VS Code is running.

I am also writing through the engineering decisions behind it: agent interaction, local-first
architecture, private connectivity, and the practical edges of building on VS Code and Copilot.

Right now, the focus is making the first-run experience straightforward across local VS Code, dev
containers, WSL, and Codespaces, then documenting what the system does and does not trust.
